# 项目指令

## 项目记忆

### 对话开始时

1. 读取主记忆文件 `.claude/project-memory.md`（如存在）
2. 扫描 `.claude/sessions/*.md`（未合并的近期会话）
3. 按时间排序，重点关注最新的会话记录
4. 简要告知用户你已了解的上下文（一句话即可）

### 对话结束时

当用户说"再见"、"bye"、"结束"、"保存进度"时：
- 使用 `/project-memory` skill 保存会话

### 其他命令

| 用户说 | 动作 |
|--------|------|
| 合并记忆 | 将所有 session 文件合并到主记忆文件 |
| 查看会话 | 列出所有未合并的会话 |
